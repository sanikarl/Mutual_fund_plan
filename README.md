# Mutual_fund_plan

Project Overview
This project is a Python-based mutual fund investment planner that helps investors create, evaluate, and optimize a diversified mutual fund portfolio. Using historical financial data and key performance indicators, the tool analyzes returns, risk, and portfolio performance to support data-driven investment decisions.

Objectives
1. Analyze historical mutual fund performance
2. Calculate key metrics such as returns, volatility, and risk-adjusted returns
3. Simulate portfolio performance under different scenarios
4. Provide actionable portfolio recommendations based on risk tolerance

Tools & Technologies
1. Python
2. Pandas, NumPy
3. Matplotlib, Seaborn, Plotly
4. Yahoo Finance / Financial APIs
5. Streamlit or Dash (for interactivity)

Key Features
1. Mutual fund data collection and preprocessing
2. Portfolio construction with customizable fund allocation
3. Risk and return analysis using Sharpe & Sortino ratios
4. Monte Carlo simulation for future performance estimation
5. Interactive visualizations and portfolio dashboard

Analysis & Visualizations
1. Portfolio performance vs benchmark
2. Risk-return scatter plots
3. Portfolio composition pie charts
4. Performance heatmaps

Outcomes & Insights
1. Identifies optimal fund allocation based on risk-return profile
2. Highlights impact of diversification on portfolio risk
3. Provides investment recommendations tailored to user preferences
